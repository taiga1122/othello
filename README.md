# nuxt-ts

> My swell Nuxt.js project

## Develop

`.env`

```
ENABLE_MOCK=true
SUPPORT_IE=false
```

``` bash
$ npm install

$ npm run dev
```

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate

# keep clean code
$ npm run lint

# check the types
$ npm run typecheck
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
